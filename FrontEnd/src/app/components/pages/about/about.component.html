This is a Powerful Budgeting app built by Jack Kawell. The frontend is built with Angular and the backend consists of a webserver written with Node.js and a database running on Postgres. The whole thing is hosted in a Docker microservices fleet.

Feel free to play around with it! If you want to checkout or contribute to the source code, you can find it all here.
